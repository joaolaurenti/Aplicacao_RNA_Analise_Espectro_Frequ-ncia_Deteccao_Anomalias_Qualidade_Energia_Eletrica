# Instruções para uso dos Scripts

Todos os códigos estão comentados e são feitos na linguagem Python

1 - Use o Scrip 01-Gerador_de_Sinais.py para gerar os sinais
        (O Script não cria a pasta onde os sinais serão salvos essa pasta deve ser criada antes de rodar o script)
        
2 - O código 02-FFT.py serva somente para plotagem e estudo dos sinais criados 
        (Observe que nome da pasta que o script acessa tem que ser a mesma onde os sinais estão salvos)

3 - Para criação do arquivo CSV é usado o script 03-Criador_dataset.py
        (Ele cria o dataset na pasta onde os códigos estão salvos)
